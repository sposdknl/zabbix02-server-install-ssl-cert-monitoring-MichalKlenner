Vytvořil jsem virtual machine, pomocí ubuntu.

Nastavil jsem 2GB RAM, 2 CPU porty 22 a 80.

Poté jsem nainstaloval apache, mysql a vytvořil databázi zabbix a localhost uživatele a nainstaloval Zabbix server, poté agent2. Dále jsem konfiguroval hosta, abych ho pak mohl monitorovat na frontendu na serveru. Vše jsem dělal pomocí příkazů ve virtualu a ne pomocí skriptů.



Na stránce https://www.zabbix.com/download?zabbix=7.0\&os\_distribution=ubuntu\&os\_version=22.04\&components=server\_frontend\_agent\_2\&db=mysql\&ws=apache jsem udělal 90% celého postupu instalace frontend agenta. Poté jsem udělal monitorování sposdk stránky pomocí hosta.
